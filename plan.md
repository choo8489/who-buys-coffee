# 커피 내기 '중력 뽑기(Gravity Roulette)' 구현 계획

GitHub Pages에 배포하기 적합한 정적 웹(HTML, CSS, JS) 기반의 가볍고 재미있는 커피 내기 게임(룰렛)을 기획했습니다. 사용자가 제안한 "중력"과 "가장 먼저/늦게 떨어지는 사람 찾기" 콘셉트를 살려, 장애물이 있는 핀볼/플링코(Plinko) 형태의 화면에서 캐릭터(이름표)들이 중력을 받아 떨어지는 게임입니다.

---

## 🎯 설계 개요 (도입부)

- **주제**: 커피 내기를 위한 플레이어별 랜덤 장애물 낙하 룰렛
- **핵심 재미**: 위에서 이름표를 떨어뜨리면 무작위 장애물에 부딪히며 떨어지는 속도가 달라짐. 
- **승/패 조건**: 누가 가장 먼저 바닥에 닿는가 혹은 가장 늦게 닿는가를 앱에서 자동 판정.
- **기술 스택**: 순수 HTML5, CSS3, 웹 물리엔진(`Matter.js` - 가볍고 구현이 쉬움)
- **배포 방식**: 단순 파일 구조로 GitHub Pages 배포 최적화 (`index.html` 기반)

## 📋 핵심 기능 요구사항

1. **사용자 입력 화면**: 
   - 커피 내기에 참여할 인원들의 이름을 입력하고 추가/삭제하는 기능.
   - 직관적이고 세련된 UI (글래스모피즘, 부드러운 애니메이션 적용).
2. **게임 진행 화면**:
   - `Matter.js`를 이용한 2D 물리 엔진 캔버스 생성.
   - 핀볼 기계처럼 화면 곳곳에 고정된 장애물(원형, 다각형 등) 배치.
   - 입력한 이름 수만큼 개별 공(플레이어)을 상단 한구석이나 여러 곳에서 동시에 낙하시킴.
   - 각 공마다 아주 약간의 탄성/마찰력 차이를 둬서 결과를 매번 예측할 수 없게 함.
3. **결과 판정 및 표시**:
   - 화면 맨 아래의 결승선(마우스 센서) 영역에 닿는 순서를 감지.
   - 1등(제일 먼저 떨어진 사람) 혹은 꼴등(제일 늦게 떨어진 사람)을 모달 창으로 짜잔! 하고 표시.

## 🛠 코드 구조 제안 (Proposed Changes)

프로젝트는 프레임워크 구성을 피하고 최대한 직관적이고 가볍게 구성합니다.

### 🌐 Frontend HTML/CSS
#### [NEW] `index.html`
- 이름 입력 폼, `canvas` 도화지 영역, 결과 표시 모달로 이루어진 뼈대.
- 배경에는 우주나 추상적인 그라데이션을 깔아 '중력'의 느낌을 줍니다.

#### [NEW] `style.css`
- 웹페이지 전반의 세련된 디자인.
- 어두운 배경과 네온 불빛 효과(Dark Mode) 적용으로 프리미엄한 느낌 구축.

### ⚙️ Game Logic (JavaScript)
#### [NEW] `script.js` (UI 및 게임 로직 컨트롤)
- 이름 목록 배열 관리.
- 게임 시작 버튼 클릭 시 DOM 제어 및 `Matter.js` 엔진 초기화.
- 물리 엔진 상에서 "플레이어 물리 객체(공)" 생성.
- 바닥 충돌 이벤트(Collision)를 감지하여 순위를 기록.

#### [NEW] `matter.js` (외부 라이브러리)
- CDN을 통해 연결하거나, 안정성을 위해 프로젝트 내에 파일 다운로드. (여기서는 CDN 사용 권장)
